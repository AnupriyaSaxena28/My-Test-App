# My-Test-App
A mobile app to manage Tasks, Schedule, and Notes.  

🧩 Features 
BottomNavigationBar → Tasks | Schedule | Notes 
Tasks (ListView) 
1: Add task (Form with date &amp; time picker) 
2: Mark complete (tap) 
3: Delete task 
Schedule (TabBar) -> Today | Upcoming 
Notes (GridView) Show notes in grid
